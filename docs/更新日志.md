# SpikMorph 更新日志

本文件按 Asia/Shanghai（UTC+08:00）时间记录项目变更，最新记录位于最前。

## 2026-06-18 11:24:03 +08:00

### 训练过程与资源收尾修复

- 将 `PPO.MAX_STATE_ACTION_PAIRS` 和 `PPO.EARLY_EXIT_STATE_ACTION_PAIRS` 的默认值由浮点数改为整数，命令行可直接使用 `16384` 等整数覆盖。
- 在主进程 PPO 训练循环中加入 `tqdm` 进度条，显示 update、已完成环境步数、FPS 和可用时的最近回报。
- 短训尚未产生完整 episode 时不再对空数组求均值，消除 `Mean of empty slice` 和 `invalid value` 警告。
- 训练结束或异常退出时显式关闭向量环境和 TensorBoard writer，避免 `SubprocVecEnv.__del__` 阶段出现 `BrokenPipeError`。
- 增加 `tqdm` 运行依赖。

## 2026-06-18 10:50:08 +08:00

### SNN Transformer 轻量化阶段 A

- 新增 `configs/snn_stage_a.yaml`，启用 Spiking Transformer，并将编码器由 5 层、FFN 1024 缩减为 3 层、FFN 256。
- 保留原始默认配置作为 331 万参数基线，避免破坏旧实验和 checkpoint 的配置语义。
- 新增 `tools/report_model_size.py`，可在不启动 MuJoCo 环境的情况下计算模型参数预算并执行上限检查。
- 在容器依赖中补充与当前 `clock_driven` API 匹配的 `spikingjelly==0.0.0.0.12`。
- 阶段 A 理论总参数量为 810,899，较原始 3,314,579 参数减少约 75.54%。

### 文档

- 新增 `docs/SNN_Transformer轻量化规划.md`，记录参数分析、分阶段结构方案、实验矩阵和验收条件。
